# MDC Partners

<p>
Back-end application created for a development test comprising 8 tests, including: Calculator, Prime Number, Factorial, Palindrome, Table, Vowel Counter, Grade Average, Interest Calculation.
</p>



### :hammer: Requirements

Docker or Node.js
